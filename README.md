# DmarcInspector

Looks up DMARC policy of a host

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'dmarc_inspector'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install dmarc_inspector
